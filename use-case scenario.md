                
## Mashup Web Application 

## Input field
- postcode 

## School 
- government school
- non-government school   


## Output
- government school   
  - school gender
  - street
  - selective school entry score (click to show graph)
  - attendance rate (click to show graph)
  - specific information 
    - schooling
    - suburb
    - postcode
    - code
    - phone
    - email
    - fax
    - student number
    - special support infomation
- non-government school   
  - school gender
  - street
  - specific information
    - schooling
    - suburb
    - postcode
    - website

## Involved outside API
--> Google Map API
